1. Nama Projek		: Let's Learn (e-learning)

2. Deskripsi Projek	:
Perkembangan berbagai media pembelajaran ini seiring dengan adanya kemajuan teknologi yang semakin pesat. Dinamika teknologi saat ini mencapai akselerasi yang luar biasa. Teknologi yang dipelajari beberapa tahun yang sudah lalu mulai tergantikan dengan teknologi yang baru termasuk berbagai cara pembelajaran secara konvensional.
Bentuk perkembangan teknologi informasi yang dapat dimanfaatkan sebagai media pembelajaran adalah menggunakan e-learning. E-learning merupakan inovasi yang dapat dimanfaatkan dalam proses pembelajaran, tidak hanya dalam penyampaian materi pembelajaran tetapi juga perubahan dalam kemampuan berbagai kompetensi peserta didik. 
Dengan itu, kami membuat web e-learning bernama Letâ€™s Learn yang merupakan salah satu upaya kami untuk mempermudah pelajar-pelajar di Indonesia yang ingin belajar lebih tentang multimedia, seperti UI/UX, Graphic Design, 3D Design, dan Video Editing.  Kita juga menyediakan 18 guru yang siap membimbing kalian.

5. Cara membuka dan menginstall projek :
   1.	Buka google chrome
   2.	Ketik pada search bar localhost/e_learning/front
   3.	Anda akan diarahkan pada menu home
   4.	Lalu pilih register dinavbar web dan isi data kalian
   5.	Setelah anda melakukan register, anda bisa melakukan login

   Login (User)
   1.	Tampil menu home
   2.	Lalu kalian bisa memilih fitur-fitur yang ada
   3.	Lalu kalian bisa memilih class dan baca konten yang anda inginkan
   4.	Jika sudah dibaca maka anda bisa memilih add to cart
   5.	Lalu kalian bisa membayarnya

   Login (Admin)
   1.	Tampil menu dashboard 
   2.	Lalu anda bisa memilih fitur-fitur yang ada
   3.	Di menu kategori kalian bisa menambahkan menghapus dan mengedit kategori
   4.	Di menu class kalian bisa menambahkan menghapus dan mengedit class
   5.	Di menu blog kalian bisa menambahkan menghapus dan mengedit konten
   6.	Di menu account kalian bisa melihat data user
   7.	Di menu order kalian bisa melihat data user yang sudah order
